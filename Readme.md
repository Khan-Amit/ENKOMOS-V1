# ENKOMOS

**Energy & Keystone Operating Management Operating System**  
*Autonomous Greenhouse & Aquaculture Production Software*

---

## Overview

ENKOMOS is a complete AI-driven software system for off-grid, renewable-energy-powered greenhouse and aquaculture production. It integrates:

- **Climate control** (temp, humidity, CO₂, VPD)
- **Structured/charged water monitoring** (ORP, pH, DO, EC)
- **Energy management** (solar, wind, biogas, compost heat, sodium-ion battery)
- **Crop & fish databases** (fully user-editable)
- **Regional presets** (tropical, temperate, arid, cold)
- **Manual override** (digital sliders + physical panel)
- **Frequency logging** (for growth research)
